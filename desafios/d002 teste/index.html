  1. Em HTML5, temos vários tipos de lista. Qual dos itens a seguir não é um tipo de lista suportado pela linguagem?

    1. Lista ordenada
    2. Lista não orenada
    3. Lista semi ordenada
    4. Lista de definição

  2. Na constução das listas ordenadas e nas não ordenadas, o uso de uma das tags se repete nos dois casos. Que tag é essa?

    1. <ul>
    2. <ol>
    3. <li>
    4. <dl>

  3. Para criar uma lista ordenada, devemos limitar todos os seus itens dentro de um par único de tags:

    1. <ol> e </ol>
    2. <ul> e </ul>
    3. <dl> e </dl>
    4. <dt> e </dt>
